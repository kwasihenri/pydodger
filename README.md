# PyDodger

A simple 2D obstacle dodging game built with Python and Pygame. The player controls a character that must avoid falling obstacles. The goal is to survive as long as possible to get a high score.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

You need to have Python and Pygame installed on your system.

- **Python:** You can download it from [python.org](https://www.python.org/downloads/).
- **Pygame:** You can install it using pip:
  ```bash
  pip install pygame
  ```

### Cloning

Clone the repository to your local machine using the following command:

```bash
git clone https://github.com/kwasihenri/PyDodger.git
cd PyDodger
```
